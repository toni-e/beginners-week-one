HTML Cheatsheet

What is HTML?

"HTML" - Hyper Text Markup Language that describes the structure of web pages.

HTML documents are described by HTML tags, and each tag describes a different element.

What is an element?

Usually an element consists of 3 parts:

  1. opening tag
  2. content
  3. closing tag

The forward slash "/" in the closing tag differentiates it from the opening tag. Without this the element isn't closed properly and this can cause difficulties later on in the structure of the webpage.

It's worth noting that not all tags need closing, an example of this is the br tag. The br tag can be used to create blank lines and is called a void tag. This tag only requires an opening tag <br>.

In HTML headings are marked with using heading tags such h1. There are six levels of heading supported. The biggest is h1 which is used for page titles. The smallest is h6. On most pages you probably won't see h4-6 used.
